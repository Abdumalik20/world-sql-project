# 🌍 World Database SQL Analysis Project

## About This Project
This is my first SQL portfolio project. I solved 10 real-world data analyst tasks using MySQL.

## What I Used
- MySQL 8.0
- World database (countries, cities, languages)

## The 10 Tasks I Solved

| # | Task | What I Learned |
|---|------|----------------|
| 1 | Capital population concentration | JOINs and percentage calc |
| 2 | English vs non-English countries | Conditional aggregation |
| 3 | Top 3 cities per continent | Window functions (ROW_NUMBER) |
| 4 | Life expectancy by government | GROUP BY with HAVING |
| 5 | Language diversity | COUNT DISTINCT |
| 6 | Independence era comparison | CASE statements |
| 7 | Cities with same name as country | Data quality checks |
| 8 | Regional density leaders | CTEs |
| 9 | Monarchy vs Republic | Pattern matching (LIKE) |
| 10 | GNP per capita ranking | RANK and PARTITION BY |

## How to Run My Queries

1. Download the world database
2. Import it into MySQL
3. Run any query from the `queries` folder
